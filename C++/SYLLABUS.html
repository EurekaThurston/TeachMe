C++ for VFX TAs · 课程总览

从零基础到「读懂并改」真实 C++ 代码

一条 18 节的完整路线。每节短小、一个收获、紧扣使命。建议顺序学,但你也可以随时跳到任意一节。

目标(使命) 能打开真实的 C++ 文件(HDK 插件、引擎模块、渲染器源码),逐行读懂它在做什么,并对它做安全的小修改、编译运行。详见 MISSION.md。

进度:18 / 18 节已产出 ✓  ·  资源:RESOURCES.md  ·  学习记录:learning-records/

第一部分 · 读懂最小单元
温和上手:一个函数怎么读、怎么在脑子里跑一遍。
  1. 01读懂函数签名 — 返回类型 / 函数名 / 参数 上手
  2. 02追踪函数体里的数据流 — if / 比较 / return 提前退出 上手
第二部分 · 语言的零件
变量、运算、循环——读任何代码都绕不开的基本构造。
  1. 03变量、类型与赋值 — 类型标签 / = vs == / const / float vs double 深入
  2. 04运算符与表达式 — 算术 / 逻辑 / 三元 / 整数除法陷阱 深入
  3. 05循环 —— 逐点处理的引擎 — for / while / range-based for 深入
第三部分 · 指针墙
阅读真实 C++ 的最大门槛。攻下这三节,代码就不再劝退。
  1. 06引用 references — T& / const T& / 为什么到处是 const& 深入
  2. 07指针 pointers — T* / & 取地址 / * 解引用 / . vs -> 深入
  3. 08内存与生命周期 — 栈 vs 堆 / new·delete / 智能指针 深入
第四部分 · 聚合数据与标准库
代码里成串、成包、成对象的数据长什么样。
  1. 09数组与 std::vector — 点/属性数组的样子 深入
  2. 10std::string 与常见标准库类型 — string / pair / map 深入
  3. 11struct 与 class 读法 — 成员 / 方法 / . 访问 / public·private 深入
  4. 12class 深入 — 构造·析构 / this / 继承 / virtual·override 深入
第五部分 · 让整个文件可读
把零散语法拼成一个真实工程的那些"胶水"。
  1. 13头文件与编译单元 — .h vs .cpp / #include / 编译·链接 深入
  2. 14命名空间与作用域解析 :: — std:: / Class::method / 库前缀 深入
  3. 15模板作为读者 —— 别被 <> 吓到 — std::vector<float> 怎么读 深入
  4. 16枚举、switch 与位标志 — enum / switch / FLAG_A | FLAG_B 深入
第六部分 · 收成(把一切串起来)
用真实 Unreal C++ 源码验收阅读力,再走通"改 → 编译 → 验证"的闭环。
  1. 17通读真实 Unreal Actor 源码 — 把第 1–16 课全用上 压轴
  2. 18改并编译 — Live Coding / 读懂报错 / 在编辑器验证 压轴
学完之后 这条路线打通的是当前使命:读懂 + 小改 Unreal C++ 代码(自定义 Actor/Component、引擎模块、Niagara)。再往后可以分支到从零写一个完整插件/组件,或深入 Niagara/渲染模块源码。到时我们更新 MISSION.md 再排新路线。
