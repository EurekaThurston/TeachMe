# C++ for VFX TAs — Resources

## Knowledge

- [LearnCpp.com](https://www.learncpp.com/)
  最高质量的免费 C++ 系统教程,28 章,每章带复习测验。被 Reddit / C++ Discord 公认为初学者首选。Use for: 任何语言核心概念(类型、函数、指针、类、内存),作为权威参考逐章对照。
- [cppreference.com](https://en.cppreference.com/)
  C++ 标准库与语言的权威参考手册。**不是教程**,是查阅工具。Use for: 看代码时遇到某个函数/类型,查它的精确定义和签名。
- [The Cherno — C++ series (YouTube)](https://www.youtube.com/playlist?list=PLlrATfBNZ98dudnM48yfGUldqGD0S4FFb)
  高质量视频系列,讲解直观,适合视觉学习者。Use for: 想看人"动手演示"某个概念(指针、内存、编译过程)时。
- [SideFX HDK: Getting Started](https://www.sidefx.com/docs/hdk/_h_d_k__intro__getting_started.html)
  Houdini 开发工具包官方入门。包含 `standalone/geoisosurface.C` 这个最简示例程序。Use for: 把 C++ 知识落到 Houdini 的真实代码上。
- [SideFX HDK: Main Docs](https://www.sidefx.com/docs/hdk/)
  HDK 完整文档(算子、编译、构建工具 hcustom)。Use for: 读懂/修改 Houdini 插件源码时的参考。

## Wisdom (Communities)

- [r/cpp_questions](https://reddit.com/r/cpp_questions)
  专门解答 C++ 初学者问题、且对新手友好的子版块(区别于偏专家的 r/cpp)。Use for: 卡住时贴出你看不懂的代码片段求解释。
- [SideFX Forum — HDK section](https://www.sidefx.com/forum/)
  Houdini 开发者社区,HDK 相关问题在这里有人答。Use for: HDK 编译报错、API 用法、真实插件问题。
