# Mission: C++ for VFX Technical Artists

## Why
我是特效向 TA,日常在 Houdini、Unreal 和独立工具/渲染器里工作。很多强大的功能(HDK 插件、引擎源码、自定义节点/deformer)都锁在 C++ 背后。学会 C++ 能让我突破"只能用别人做好的工具"的天花板,真正读懂并定制这些底层代码。

## Success looks like
- 能打开一个真实的 C++ 文件(HDK sample、Unreal 模块、渲染器源码),逐行看懂它在做什么
- 能对现有 C++ 代码做小修改并成功编译运行(改参数、加分支、调逻辑)
- 不再因为"这是 C++ 写的"就放弃定制某个工具

## Constraints
- 几乎零正式编程基础(主要靠节点图、可视化工具工作,可能接触过一点 VEX/表达式)
- 学习以**阅读+修改**为核心,不追求从零造大型工程
- 时间有限,每节课要短、要能快速完成、要有立刻能用上的收获

## Out of scope (暂时)
- 大型 C++ 工程架构/设计模式
- 手写复杂模板元编程
- 从空白文件独立开发完整插件(先能读懂改动,再谈从零写)
