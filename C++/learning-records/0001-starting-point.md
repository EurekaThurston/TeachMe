# 起点:几乎零编程基础,目标是"读懂并改"

用户自评几乎零正式编程基础,但身份是特效向 TA,横跨 Houdini / Unreal / 独立渲染器,因此几乎确定有节点图、属性(attribute)、几何/向量、着色等领域直觉,可能接触过一点 VEX/表达式。三个月目标是**读懂现有 C++ 代码并做小修改**,而非从零造工程。

**Implications for teaching:**
- 起点要温和,假设不懂指针、类型系统、编译流程。
- 大量使用 VFX 类比(向量、属性、clamp/fit、节点输入输出)作为桥梁——这是用户已有的最强直觉。
- 课程重心放在**阅读技能**(看懂签名、追踪类型与数据流)优先于从零写代码。
- 每课要极短,working memory 小。
