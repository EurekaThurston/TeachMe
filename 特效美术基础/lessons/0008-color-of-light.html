光的颜色:暖光冷影

明度讲完了亮暗,这一课讲光的「颜色」。一个口诀贯穿全程:暖光配冷影、冷光配暖影。暗部从来不是死黑。

本课目标(一个收获) 你能用冷暖关系而非单纯亮暗来安排特效的光与影:主光暖则暗部偏冷、主光冷则暗部偏暖,并让发光特效给它照到的表面染上自己的光色,让特效"属于"场景。

一、光是有颜色的(color temperature)

我们常默认光是"白"的,其实光有色温 color temperature:从暖橙(烛火、夕阳、钨丝灯)到冷蓝(阴天天光、月光、正午阴影里的天空光)。 同一个物体,在暖光下偏橙黄、在冷光下偏青蓝——固有色没变,是光的颜色染了它。读光色,要看物体的亮面。

二、暖光冷影 / 冷光暖影

这是 Gurney 反复强调的核心关系:主光是暖的,阴影就偏冷;主光是冷的,阴影就偏暖。 为什么?因为阴影并非"没有光",而是被环境里另一种光填充——通常是和主光相反色温的天光/反弹光。于是亮暗之间不只有明度差,还有一道冷暖张力,画面立刻"活"起来,而不是单色的明暗。

拖动色温滑块。注意:你只在调光色,但球的暗面会自动偏向相反色温:

亮面:暖光(暖橙)
暗面:→ 冷影(偏蓝)

暖光 → 冷影、冷光 → 暖影。亮暗之间多了一道冷暖关系,比纯灰的明暗有层次得多。

三、环境光与反弹光:阴影不是死黑

紧接上一点:阴影从来不是纯黑。它被周围反弹回来的光(bounce light)和天光(ambient)染色——草地旁的阴影偏绿,蓝天下的阴影偏蓝,暖墙边的阴影带暖。 特效也一样:你的特效暗部应该带一点环境色,而不是一抹纯黑。死黑会让特效"挖了个洞"、和场景脱节。

四、特效的光色外溢

这是把上面三点落到 VFX 的关键动作——让特效"属于"场景:

  • 染光 color spill:一团蓝色能量,应该在场景里打出蓝光——它脚下的地面、旁边的烟、附近角色的接触面,都该接收这层蓝色。否则蓝能量和场景"各过各的"。
  • 暗部补互补:特效的暗部别只压黑,掺一点互补的暖色(蓝特效暗部带点暖),制造冷暖层次,暗部立刻有"空气"。
  • 接触面最明显:特效与表面接触、最近的地方,染光最强;远处衰减(回扣 0005 平方反比)。
一句话判断 看一个发光特效合不合场景,就看:它有没有把自己的颜色"漏"到周围去?蓝火却没在地上留一点蓝、暗部是纯黑——它就还"飘"在场景之上,没融进去。

五、接回 Niagara

  • 光组件颜色 = 特效主色:Light Renderer / 光组件的颜色设成特效的主色(蓝能量给蓝光),让它向场景投出对的色光。
  • 给周围 emissive / decal 染特效色:接触地面用染了特效色的 decal(贴花)或 emissive,做出"光漏到地上"。
  • 烟雾 / 暗部 color 掺环境 / 互补色:烟和暗部的颜色别用纯黑/纯灰,掺入环境色或互补色温。
  • Color-over-life 用不同色温:亮处与暗处、起手与消散,用不同色温(亮暖暗冷或反之),而不是同一个色相只改明度。

六、常见误区

  • 暗部纯黑 / 纯灰:没有冷暖氛围,特效像挖了个洞,和场景脱节。
  • 发光特效不给环境染色:蓝火不在地上留蓝光,特效"飘"在场景上,不属于这里。
  • 亮暗同色温:亮面暗面一个色相只改明度,画面"平"、没有冷暖张力。
