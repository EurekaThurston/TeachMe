# 起点:零散美术理论基础，目标是"读得清 + 更好看 + 能评判"

用户是 3D 游戏特效师(主力 Unreal/Niagara)，技术能落地但缺美术底层语言。自评美术理论**零散**——听过互补色、冷暖等名词但串不起来、用不熟。核心诉求(经访谈确认)按优先级:① 特效在游戏里**读得清** ② 特效**更好看、更有冲击力** ③ 能用美术语言**评判并指导**他人作品。

**Implications for teaching:**
- 起点温和，假设没有系统的色彩/明度框架，但**有极强的 VFX 直觉**(发光、叠加、渐变、HDR、颜色/时间曲线)——这是最强的已有锚点，大量用作类比。
- 教学顺序按经典 **value-first**:先把明度/黑白灰(可读性的地基)讲透，再进色彩。这直接服务第一优先级"读得清"。
- 重交互与即时反馈(去色测试、看图判断、配色判断)，轻纯阅读——契合动手型学习者偏好。
- 每个概念都要落到 Niagara 的具体可操作处，避免"太抽象"。
- 每课极短，working memory 小。
