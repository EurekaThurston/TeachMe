3D 数学 for VFX · 第 8 课 · 模块 1 向量

叉积 cross:造一个垂直方向

点积给你一个数,叉积给你一个向量——而且这个向量同时垂直于输入的两者。算三角面法线、搭切线空间、判左右手,全靠它。这是模块 1 的收尾,也是通往「空间与变换」的桥。

本课目标(一个收获) 你能说清叉积 a × b 的两件事:方向——同时垂直于 a 和 b(即垂直于它们所在的平面);长度——等于 a、b 张成的平行四边形面积 |a||b|sinθ。会用它算面的法线,并知道叉积不可交换(a×b = −b×a),方向由手性决定。

叉积给的是垂直向量

给两个 3D 向量 a、b,a × b 是一个新向量,同时垂直于 a 和 b。换句话说,a 和 b 决定一个平面(像一张纸),叉积的结果是垂直戳出这张纸的那支箭。这就是它最重要的用途:已知面上两条边,求这个面朝哪——求法线。

a × b = ( ay·bz − az·by ,　az·bx − ax·bz ,　ax·by − ay·bx )

这个公式不用背(需要时查术语表或让我给)。要记的是它的含义:输出垂直于输入两者。2D 里没有「第三个垂直方向」,所以 2D 叉积退化成一个标量 ax·by − ay·bx——它就是上面公式的 z 分量,代表「平行四边形的有向面积」,也告诉你 b 在 a 的左边还是右边(正=逆时针/左,负=顺时针/右)。

这里显示 2D 叉积(z 分量)。正=b 在 a 逆时针侧,负=顺时针侧,0=共线。它的绝对值=两箭张成的平行四边形面积。

长度 = 面积,也 = sinθ

叉积的长度 |a × b| = |a||b|sinθ,几何上正是 a、b 张成的平行四边形面积。和点积对照着记最清楚:

点积 dot叉积 cross
结果标量(数)向量(3D)
公式核心|a||b|cosθ|a||b|sinθ
最大时两者平行(θ=0)两者垂直(θ=90°)
为 0 时垂直(θ=90°)平行 / 共线(θ=0 或 180°)
量的是多同向多垂直 / 围出多大面积

所以:两个向量越垂直,叉积越长;越平行,叉积越短(平行时为零向量——平行的两条边张不出面积,也定不出唯一的垂直方向)。这和点积正好互补:点积在平行时最大、垂直时为零。

杀手级应用:从三个点算三角面法线

网格的每个三角形由三个顶点 P0、P1、P2 组成。它朝哪?用两条边叉一下: 

float3 e1 = P1 - P0;            // 边 1(终点减起点,第 5 课)
float3 e2 = P2 - P0;            // 边 2
float3 N  = normalize(cross(e1, e2)); // 垂直于面 → 法线,再归一化成纯方向

三步用上了整个模块 1:减法求边、叉积求垂直、归一化成单位法线。这就是「几何法线」的来历。如果法线朝反了(朝模型内部),把 cross(e1, e2) 换成 cross(e2, e1) 翻面即可——因为 a×b = −b×a。

手性决定方向(回到第 1 课) 叉积指向哪一侧,由坐标系的手性决定。UE 是左手系,很多教程是右手系——这就是为什么你抄来的法线/切线公式有时朝反。记住:方向反了先怀疑手性或叉积参数顺序,翻一下 cross 的两个参数顺序通常就好。

搭正交基:为模块 2、3 铺路

叉积还能从一个方向造出一整套互相垂直的坐标轴(正交基)。比如已知粒子的运动方向 forward,想给它配一套「右、上」轴(让它朝运动方向喷射 / 拉伸): 

float3 fwd   = normalize(velocity);
float3 right = normalize(cross(worldUp, fwd)); // 垂直于 up 和 fwd → 指向右
float3 up    = cross(fwd, right);            // 再叉一次补齐第三轴,已自动正交

这三根互相垂直的单位向量,就是一个坐标空间的基(下一模块的主角),也是 look-at 朝向构造(第 18 课)和切线空间 TBN(第 14 课)的核心套路。你现在已经具备搭它的全部零件了。

接回引擎 Niagara 让 Sprite/Ribbon「朝速度方向对齐」、Mesh 粒子「看向运动方向」,内部都在用叉积从 velocity 构造朝向基。你理解了这套,就能看懂(并自定义)那些 Align 模块到底在算什么,而不是只会勾选项。

常见误区

  • 以为叉积可交换。 a×b = −b×a,换序会翻面。法线朝反时,交换叉积两个参数即可。
  • 对平行向量求叉积。 平行(或共线)时叉积是零向量,normalize 它会得 NaN。求法线时若两边共线(退化三角形)要防一手。
  • 把叉积结果当标量(或把点积当向量)。 3D 叉积给向量,点积给标量。要垂直方向用 cross,要同向程度用 dot,别用混。
  • 忽略手性导致方向反。 UE 左手系 vs 教程右手系。搬公式后法线/切线朝反,先查手性与参数顺序。
