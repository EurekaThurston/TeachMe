术语表 Glossary

本门的统一术语。每节课都遵守这里的叫法与符号约定;有歧义时以本表为准。随课程增长。

符号约定 向量用粗体小写 a、b 或带分量 (x, y, z);标量用普通字母 k、t。HLSL 里 float2/3/4; Niagara 里向量是 Vector / Position。UE 坐标系:左手系,X=前、Y=右、Z=上,单位 cm;角度对外用度(°),三角函数内部用弧度(rad)。

模块 0 · 地基

坐标系 coordinate system
用一组数(坐标)给空间里每个点编号的规则。需要:一个原点 origin + 几条带方向和刻度的轴 axis。
左手 / 右手系 left/right-handed
三条轴的朝向手性。决定叉积方向和「正旋转」方向。UE 是左手系;OpenGL/多数数学书是右手系。
弧度 radian
角的「自然」单位:半径为 1 的圆上,转过的弧长就是弧度。整圈 = 2π rad = 360°。三角函数吃的是弧度。
sin / cos
单位圆上转过角 θ 后,落点的 y 坐标 = sin θ,x 坐标 = cos θ。范围 [−1, 1],周期 2π。
atan2(y, x)
由一个方向 (x, y) 反求它的角度,自动处理四个象限。求朝向角的标准工具。
lerp(a, b, t) linear interpolation
线性插值:a + (b − a)·t。t=0 给 a,t=1 给 b,中间按比例混合。HLSL lerp。
clamp / saturate
把数限制进区间。clamp(x, lo, hi);saturate(x) = clamp 到 [0, 1],shader 里极常用。
smoothstep(a, b, x)
把 x 从 [a, b] 平滑映射到 [0, 1],两端是平滑的 S 形(导数为 0),不像 lerp 那样硬。做软边/渐变。
frac(x)
取小数部分:x − floor(x)。把任意数折回 [0, 1)。UV 平铺、随机、重复图案的基石。

模块 1 · 向量

向量 vector
一组有序的数 (x, y, z)。可表示位置(从原点出发的点)或方向+大小(一支箭)。同一组数,看你怎么用。
标量 scalar
单个数,没有方向。用来缩放向量。
分量 component
向量的某一维:a.x、a.y、a.z。HLSL 里可 swizzle:a.xy、a.xxz。
长度 / 模 length / magnitude
箭有多长:|a| = √(x² + y² + z²)(勾股定理推广)。HLSL length(a)。
归一化 normalize
把向量除以自己的长度,得到长度=1、方向不变的单位向量。用来表示「纯方向」。HLSL normalize(a)。
单位向量 unit vector
长度恰为 1 的向量。法线、光照方向、朝向都该是单位向量。
点积 dot product
a·b = ax·bx + ay·by + az·bz = |a||b|cosθ。结果是标量。测「多同向」:>0 同侧、0 垂直、<0 反向。HLSL dot(a,b)。
叉积 cross product
两个 3D 向量给出同时垂直于二者的新向量,长度=两者张成平行四边形面积=|a||b|sinθ。算法线、建正交基。HLSL cross(a,b)。

模块 2 · 空间与变换

坐标空间 coordinate space
一套「原点 + 基」。常见:Local(物体自身)、World(全局)、View(相机)、Tangent(表面切线)、Screen/Clip。同一个点在不同空间坐标不同。
基向量 basis vector
定义一个空间的几条轴向量。任何向量都是基向量的加权和(线性组合)。
矩阵 matrix
一个数表;几何上=「装着变换后基向量的盒子」。矩阵 × 向量 = 把向量从一个空间搬到另一个空间。
齐次坐标 homogeneous coordinate
给 3D 点补第 4 维 w:点用 w=1、方向用 w=0。这样 4×4 矩阵才能同时表达旋转/缩放和平移。
TRS
Translate(平移)·Rotate(旋转)·Scale(缩放)三种基本变换,组合成物体的世界变换。乘法顺序有讲究。
逆转置矩阵 inverse-transpose
变换法线要用的矩阵(模型矩阵的逆的转置)。非均匀缩放下,直接用模型矩阵会让法线歪掉。
切线空间 / TBN
贴在表面上的局部坐标系:Tangent、Bitangent、Normal 三轴。法线贴图存的是切线空间法线(所以整体偏蓝)。

模块 3 · 旋转

欧拉角 Euler angles
用 pitch/yaw/roll 三个绕轴角描述朝向。直观但有万向锁 gimbal lock、且依赖轴序。
万向锁 gimbal lock
欧拉角在某些姿态下两个轴重合、丢失一个旋转自由度的现象。
四元数 quaternion
用 4 个数 (x, y, z, w) 紧凑编码「绕某轴转某角」。无万向锁、好插值。UE 里是 FQuat。
slerp
球面线性插值:在两个朝向之间匀速、最短地转过去。朝向过渡用它,别用欧拉角 lerp。

模块 4 · 程序化 / shader 数学

UV
贴图/材质上的 2D 坐标,通常 [0,1]²。乘缩放=平铺,加偏移=移动,配合 frac 重复。
SDF 有向距离场 signed distance field
一个函数:给点返回「到形状边界的距离」,内部为负、外部为正。配 smoothstep 出软边、配 min/max 做布尔。
噪声 noise
看似随机、实则可复现的连续函数(value/Perlin/simplex)。做扰动、烟、火、消融。
极坐标 polar coordinates
用 (半径 r, 角度 θ) 而非 (x, y) 描述平面点。做径向/旋涡 UV。r=length(uv),θ=atan2(uv.y, uv.x)。
