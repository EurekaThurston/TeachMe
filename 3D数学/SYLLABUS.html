3D 数学 for VFX TA

课程总纲 · 从零到能自己推

为 Unreal/Niagara 特效 TA 设计。几何直觉优先,代码锚点 HLSL + Niagara 并重。一条从「坐标是什么」直达「SDF / 四元数 / 切线空间」的路。

怎么用这份总纲 按顺序学,每节是一个独立 HTML、十几分钟读完 + 一组即时反馈测验。■ 绿色=已出可学,■ 灰色=规划中。 每课都标了它服务的场景:◆HLSL 偏材质/shader、◆Niagara 偏粒子矢量。术语统一见 术语表。

模块 0 · 地基

坐标 / 角度 / 数值管道

在碰向量之前,先把三件「天天见但没人正经讲」的地板补平:空间怎么编号、sin/cos 到底是什么、shader 里那些 lerp/saturate/frac 是什么。

  1. 坐标系与空间 ◆HLSL ◆Niagara 收获:能说清「一个点 = 原点 + 几条带刻度的轴」,看懂 UE 左手系 X前Y右Z上,UV 是 2D 坐标。
  2. 角度与三角:sin/cos 与单位圆 ◆HLSL 收获:在单位圆上看懂 sin=y、cos=x,弧度 vs 度,理解 shader 里的波动/脉冲为何全是 sin。
  3. 数值管道:lerp / clamp / saturate / smoothstep / frac ◆HLSL ◆Niagara 收获:掌握把一个数映射成另一个数的五个基本工具——TA 每天写的「数值管道」。

模块 1 · 向量

一切的心脏

向量是 3D 数学的核心。点、方向、速度、法线、颜色——全是向量。这一模块是后面所有内容的地基,慢慢嚼透。

  1. 向量是什么:位置 vs 方向 ◆Niagara ◆HLSL 收获:同一组数 (x,y,z) 何时是「点」何时是「箭」,float2/3/4 与 swizzle。
  2. 加减与缩放:位移、速度×dt ◆Niagara 收获:向量加=首尾相接,减=指向差,标量乘=伸缩;看懂 Position += Velocity·dt。
  3. 长度与归一化:纯方向 ◆HLSL ◆Niagara 收获:length 是勾股定理,normalize 给长度=1 的纯方向;明白法线/光向为何必须归一化。
  4. 点积 dot:同向程度与光照 ◆HLSL 收获:dot=|a||b|cosθ,读懂 N·L 兰伯特光照、菲涅尔、用 dot 判同向/垂直。shader 的灵魂。
  5. 叉积 cross:垂直与法线 ◆HLSL ◆Niagara 收获:cross 给同时垂直的向量、长度=面积;算三角面法线、建正交基、判手性。

模块 2 · 坐标空间与变换

TA 最常翻车处

「东西显示在错的位置 / 朝向歪了 / 法线贴图发黑」——根因几乎都在坐标空间和矩阵。这一模块把它们彻底讲穿。

  1. 坐标空间:Local / World / View / Tangent ◆Niagara ◆HLSL 收获:同一个点在不同空间有不同坐标;Niagara 的 Local vs World、材质里的空间转换节点。
  2. 基向量:一个空间 = 原点 + 三根轴 ◆HLSL 收获:任何向量都是基的线性组合——这是理解矩阵的钥匙。
  3. 矩阵是什么:装着变换后基向量的盒子 ◆HLSL 收获:矩阵×向量=换空间;4×4 与齐次坐标(为什么要第 4 维 w)。
  4. TRS 与乘法顺序:变换链 ◆HLSL ◆Niagara 收获:平移/旋转/缩放矩阵,为什么乘序不能换、为什么从右往左读。
  5. 法线为什么用逆转置矩阵 ◆HLSL 收获:非均匀缩放下法线会歪——inverse-transpose 的由来。TA 高频坑。
  6. 切线空间与法线贴图:TBN ◆HLSL 收获:法线贴图为什么是蓝的、世界空间 vs 切线空间法线、TBN 矩阵怎么搭。

模块 3 · 旋转

朝向与四元数

让粒子朝向运动方向、让物体平滑转向、避免万向锁——旋转是 TA 绕不开的硬骨头。用前面的 sin/cos 与 cross 一步步搭上去。

  1. 2D 旋转与旋转矩阵:转 UV ◆HLSL 收获:用 sin/cos 推 2D 旋转矩阵,在材质里旋转贴图/UV。
  2. 3D 旋转、欧拉角与万向锁 ◆Niagara 收获:pitch/yaw/roll、轴序的坑、万向锁到底锁了什么。
  3. 四元数:紧凑、无锁、好插值 ◆Niagara 收获:四元数是「轴+角」的编码,为什么用它,slerp 平滑过渡朝向,UE 的 FQuat。
  4. Look-at:从方向向量构造朝向 ◆Niagara ◆HLSL 收获:用 cross 建正交基,让粒子/物体看向目标——把模块 1 和 3 接起来。

模块 4 · 程序化 / shader 数学

把前面全用起来

收口模块:UV、噪声、SDF、时间波动、缓动曲线——把向量、三角、数值管道全部组合成真实的程序化特效。学完能读懂 Quilez / Book of Shaders。

  1. UV 与采样的数学:平铺 / 偏移 / 极坐标 ◆HLSL 收获:tiling、frac 重复、旋转 UV、极坐标做径向/旋涡。
  2. 噪声与随机:hash、value、Perlin 直觉 ◆HLSL 收获:为什么 frac(sin·大数) 是随机,噪声怎么做扰动/消融/烟火。
  3. SDF 有向距离场:形状即函数 ◆HLSL 收获:圆/盒 SDF、smoothstep 软边、min/max 布尔、smooth min。读懂 Inigo Quilez。
  4. 时间、波与脉冲:用 sin 做动画 ◆HLSL ◆Niagara 收获:相位/频率/振幅、呼吸与脉冲、panner、time 驱动的运动。
  5. 曲线与缓动收尾:把数值管道串成动画 ◆HLSL ◆Niagara 收获:ease 家族、smoothstep 再访、Bezier 直觉,串起一条完整的程序化动画管线。
